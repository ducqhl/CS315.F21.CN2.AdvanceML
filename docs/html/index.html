Tổng quan hệ thống

Hệ thống thu thập giá Bitcoin và Dogecoin từ CoinGecko API, xử lý song song theo hai luồng — Speed Layer (Kafka + Spark Streaming, latency ~30s) và Batch Layer (Spark Batch, xử lý 4,165 ngày lịch sử) — hợp nhất kết quả tại Serving Layer (MongoDB), rồi huấn luyện mô hình LSTM dual-head để dự báo giá 7 ngày tới.

Tại sao Lambda, không phải Kappa? Ba lý do chính: (1) chi phí reprocessing SMA-90 trên 4,165 ngày, (2) LSTM cần StandardScaler nhất quán, (3) hai loại query có SLA khác nhau. Chi tiết →

Các thành phần hệ thống

🏗️ Lambda Architecture

Thiết kế tổng quan, lý do chọn Lambda vs Kappa, luồng dữ liệu ba layer. Phân tích trade-off latency vs accuracy.

Architecture

📡 CoinGecko Producer

Poll mỗi 600s, acks=all, retries=3, linger_ms=100. OHLC mỗi 3 cycle. Rate limit logic, exponential backoff. Demo tier = 10k calls/month.

Kafka Ingestion

⚡ Spark Streaming

Watermark 10min, micro-batch 30s. Query A: windowed agg. Query B: per-record RSI/Bollinger/VWAP/ATR. foreachBatch writes MongoDB.

Spark Realtime

📦 Spark Batch

4,165 ngày lịch sử. Outputs: daily_stats, historical_sma (SMA-7/30/90), coin_correlation. Submitted via run_batch.sh.

Spark Batch

🗄️ MongoDB

7 collections. TTL policy trên speed layer docs. Compound indexes cho time-range queries. Single serving store cho cả batch + speed.

Database

🧠 LSTM Model

9 features · 2-layer encoder · 128 hidden units · dual-head (price regression + trend classification). Walk-forward validation, 61.1% directional accuracy (BTC).

Deep Learning

🔌 FastAPI Backend

JWT stateless auth. REST endpoints: prices, predictions, model registry, accuracy tracking. Merges batch + speed views per query. Port 8000.

API

🖥️ React + Streamlit

React 19 + TypeScript + Vite + Tailwind. 5 trang: Dashboard, Realtime, Technical, Predictions, Correlation. Streamlit dashboard port 8501.

Frontend

🧪 E2E Testing

3-layer E2E tests với testcontainers thật. Layer 1: Producer→Kafka. Layer 2: Spark Batch→MongoDB. Layer 3: ML Pipeline→MongoDB. 24 test cases.
