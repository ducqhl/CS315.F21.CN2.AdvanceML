Luồng dữ liệu End-to-End

Từ CoinGecko API đến React dashboard — từng bước trong pipeline.

1
CoinGecko Ingestion
Producer poll API mỗi 600 giây. Fetch price + OHLC 4h (mỗi 3 cycle). Serialize thành JSON và publish lên Kafka với acks=all. Song song ghi raw tick vào MongoDB live_prices.
coin="BTC" price=77473.37 change_24h=-1.24%
open=77200 high=77600 low=77100 close=77473
→ Kafka topic: crypto_raw
2
Spark Streaming — Speed Layer
Spark đọc micro-batch (trigger 30s). Watermark 10 phút. Query A: window agg 20min/5min → window_stats. Query B: tính SMA-5, SMA-20, RSI-14, VWAP-60, Bollinger Bands → realtime_prices. Alert nếu |change_24h| > 5%.
RSI(14)=58.3 VWAP=77200 SMA20=76900
BB_upper=78500 BB_lower=75300
→ MongoDB: realtime_prices
3
Spark Batch — Batch Layer
Xử lý 4000+ ngày CSV lịch sử. Tính daily_stats (OHLCV), historical_sma (7/14/30/90d), và rolling Pearson correlation 30d giữa BTC-DOGE log-returns. Chạy hàng ngày hoặc trước khi train LSTM.
daily_stats: 4136 rows BTC + ~4000 DOGE
corr_30d(BTC,DOGE) ∈ [0.3, 0.85]
→ MongoDB: daily_stats, coin_correlation
4
LSTM Training & Inference
Đọc daily_stats từ MongoDB. Engineering 9 features từ close price (log_return, momentum, RSI, VWAP...). StandardScaler fit on train only. Sliding windows 60×9 → 7. Walk-forward 6 folds. Inference Scheduler chạy mỗi 5 phút.
Input: (1, 60, 9) → h_T: (1, 128)
Price Head: 7 log-returns → USD prices
Vol Head: 7 σ values (confidence band)
5
FastAPI Serving
Client gửi credentials → nhận JWT (24h). Axios attach token vào mọi request. FastAPI query MongoDB và trả về JSON. Model Registry endpoint cho phép xem metrics và rollback models.
GET /api/predictions?coin=bitcoin&horizon=7
Authorization: Bearer eyJhbGciOiJIUzI1...
→ [{date, predicted_price, confidence_band}]
6
React Frontend / Streamlit
5 trang React: Dashboard, Realtime, Technical Analysis, Predictions (LSTM forecast + confidence band từ Volatility Head), Correlation. Streamlit dashboard kết nối trực tiếp MongoDB.
PredictionsPage: line chart + σ band
TechnicalPage: candlestick + RSI/MACD/BB
CorrelationPage: rolling heatmap BTC-DOGE
